# Homebrew Tap

Custom Homebrew tap for tools by [@danmartuszewski](https://github.com/danmartuszewski).

## Installation

```bash
brew tap danmartuszewski/tap
```

## Available Casks

### hop

Fast, elegant SSH connection manager.

```bash
brew install danmartuszewski/tap/hop
```

For more info, see the [hop repository](https://github.com/danmartuszewski/hop).
